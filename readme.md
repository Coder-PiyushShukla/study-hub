# 🚀 SmartPortal: The Intelligent Career & Placement Ecosystem

SmartPortal is an all-in-one, AI-driven platform designed to streamline the transition from academic learning to professional employment. Developed as an individual project by **Piyush Shukla**, this application replaces fragmented study methods with a centralized, high-tech environment for career preparation.

## ✨ Core Features & Functionality

### 1. ⚡ Immersive 3D Experience
* **Interactive UI/UX:** A glass-morphic, 3D-perspective login and registration system.
* **Dynamic Hover Physics:** Interactive cards that respond to mouse movement using `framer-motion` and CSS 3D transforms.
* **Context-Aware Design:** Real-time visual feedback and state-of-the-art dark mode aesthetics.

### 2. 🤖 AI Mentor Pro (Gemini Integration)
* **Resume Analysis:** Upload or paste resume content for instant AI-driven feedback.
* **Dynamic Interview Prep:** Generate role-specific interview questions (SDE, DevOps, Data Science) based on current industry trends.
* **Smart Roadmaps:** Personalized learning paths generated by AI to help students bridge skill gaps.

### 3. 📚 Smart Study Hub
* **Curated Resource Library:** Organized repository for semester-wise notes, textbooks, and verified study materials.
* **Peer Interview Experiences:** A community-driven section where students share real-world interview questions and company-specific rounds.
* **Filterable Content:** Easily sort resources by subject, department, or difficulty level.

### 4. 🏢 Placement Management System
* **Company Tracking:** Real-time updates on upcoming campus drives.
* **Eligibility Engine:** Automated checks to see if a student meets a specific company's CGPA or skill requirements.
* **Application Workflow:** One-click interest expression and status tracking for recruitment rounds.

### 5. 🛡️ Advanced Security & Architecture
* **Role-Based Access Control (RBAC):** Distinct permissions and dashboards for Students and Admins.
* **Secure Authentication:** JWT-based stateless authentication with bcrypt password hashing.
* **API Security:** Implementation of CORS, Helmet.js for header protection, and rate-limiting to prevent brute-force attacks.

---

## 🛠️ Technical Architecture

### Frontend (The Face)
* **React 18 & Vite:** For lightning-fast builds and optimized production performance.
* **Tailwind CSS:** For highly customized, responsive layouts.
* **Framer Motion:** Powering the complex 3D animations and page transitions.

### Backend (The Brain)
* **Node.js & Express:** Scalable RESTful API architecture.
* **Google Gemini SDK:** Fueling the AI-driven mentorship features.
* **Mongoose ODM:** Managing complex data relationships in MongoDB.

### Cloud & DevOps
* **Vercel:** Frontend hosting with continuous deployment.
* **Render:** Backend hosting with managed environment variables.
* **Cloudinary:** (Optional/Planned) For secure storage of student resumes and profile images.

## 💻 Local Setup & Installation

1.  **Clone the Repository**
    ```bash
    git clone [https://github.com/Coder-PiyushShukla/study-hub.git](https://github.com/Coder-PiyushShukla/study-hub.git)
    cd study-hub
    ```

2.  **Server Configuration**
    ```bash
    cd server
    npm install
    # Create .env with: MONGO_URI, JWT_SECRET, GEMINI_API_KEY, CLIENT_URL, PORT
    npm run dev
    ```

3.  **Client Configuration**
    ```bash
    cd ../client
    npm install
    # Ensure your baseURL in api.js matches your server port
    npm run dev
    ```
